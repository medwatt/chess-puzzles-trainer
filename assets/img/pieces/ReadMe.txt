This folder contains the bundled piece sets used by Chess Puzzles.

Each subfolder identifies a set and contains the images used for the various
dimensions of the board. Images can be provided in gif or png format (png
requires tcl/tk version 8.6 or version 8.5 with optional tkimg package).

Most images used here were generated from freeware chess fonts available in TrueType format at the En Passant website "Chess Fonts" page
http://www.enpassant.dk/chess/fonteng.htm
Copyright notice for the images should be specified in a copyright file within each folder.

Each picture contains all the pieces (pawn, knight, bishop, rook, queen, king)
for both colors and must comply with the specified size.
For example:
myownpieceset_40.gif must be an image with length 480 (40 x 12) and 40 pixels height.
myownpieceset_70.gif must be an image with length 840 (70 x 12) and 70 pixels height.
etc..

You can add your own set by creating a directory with your preferred images.
The new set will be available the next time Chess Puzzles starts.
